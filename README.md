# RAG Debugger

A plug-and-play evaluation layer for Retrieval-Augmented Generation (RAG) systems.

## Overview

RAG Debugger analyzes:

- Retrieval similarity
- Query coverage
- Unsupported entities and numbers (hallucination signals)
- Faithfulness score

It helps developers understand whether their RAG pipeline is grounded in retrieved context.
